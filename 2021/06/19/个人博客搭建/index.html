使用hexo + gitee page搭建个人博客

利用gitee page托管网页，省去了服务器，可以随时更新代码，非常便捷

本文参考文档 | Hexo

1、软件依赖

1、node.js

2、npm

3、git

以上软件请自行百度搜索安装

2、创建仓库

在gitee中新建一个你的码云仓库，仓库名为

https://gitee.com/你的用户名/仓库名

开启gitee page：

3、配置ssh key

1、桌面右键使用git bash打开，设置user.name和user.email配置

2、生成ssh密钥

完成后按三次回车即可

3、复制密钥

复制上面的密钥信息即可

4、添加密钥到gitee

4、安装hexo

  1. 打开git bash输入代码 等待完成后输入，查看是否安装完成
  2. 进入需要安装到的目录，初始化hexo

5、配置_config.yml文件

打开目录下的_config.yml文件，末尾补充

deploy:
type: git
repo: https://gitee.com/你的空间名/仓库名

branch: master

6、提交、部署

安装部署工具：npm install hexo-deployer-git -save

回到gitee更新页面即可看到内容
